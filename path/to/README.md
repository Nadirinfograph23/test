# مشروع Infograph23

هذا مشروع يهدف إلى عرض البيانات والمعلومات بطريقة جذابة و سهلة الفهم.

## محتويات المشروع
- `index.html`: صفحة الويب الرئيسية التي تعرض المعلومات.
- `style.css`: ملف الأسلوب الذي يُستخدم لتجميل صفحة الويب.

## كيفية التشغيل
1. قم بتحميل الملفات.
2. افتح `index.html` في المتصفح.

## الهدف
تقديم معلومات بطريقة مبتكرة وفعّالة.
